# 0x16. C - Simple Shell
  This ```alx project``` teaches students about the tools to write their simple Shell programs. 
The following are some of the resources needed for he the completion of this project

1. [Everything you need to know](https://intranet.alxswe.com/concepts/64)
2. [Approaching a Project](https://intranet.alxswe.com/concepts/350)

	List of commands to learn are as follow;
	1. PID and PPID
	2. Command line arguments
	3. Executing a program
	4. Creating processes
	5. Wait
	6. fork + wait + execve
